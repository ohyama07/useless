# 「0」から「1」を作る機械

## ひらがなの小数単位

漢字の小数単位に加え、`いちぶ`、`にりん`、`さんもう`、`いちねはんじゃくじょう`などの「数字読み＋ひらがな小数単位」を変換できます。複数の単位を指数順に並べた`いちぶにりんさんもう`のような表記も、一つの数値として処理します。

- `ぶ`は小数単位の「分」として扱います。
- `ふん`と`ぷん`は時間の単位として扱い、ひらがな小数単位には含めません。
- 単位読みだけの`ぶ`、`りん`、`し`、`び`などは検出しません。直前に数字読みが必要です。
- `いちぶ`などは、文脈上の「一部」と小数単位の「一分」を区別できません。意味解析は行わず形式規則を優先するため、「文章のいちぶ」のような一般語も小数表記として処理される場合があります。
- 通常小数と歩合表記は別体系です。通常小数では「分」が`10^-1`、涅槃寂静が`10^-24`です。歩合表記では「割」が`10^-1`、「分」が`10^-2`になります。
- `ねはんじゃくしょう`も入力できますが、出力はcanonicalな`ねはんじゃくじょう`へ統一します。
- `二割五分六厘`のような漢字の歩合表記に加え、`さんわりにぶごりん`のようなひらがなの歩合表記にも対応します。
- 正常に解析された単位列に`わり`が含まれる場合は歩合表記として、`わり = 10^-1`、`ぶ = 10^-2`、`りん = 10^-3`の指数を使用します。`わり`を含まない`にぶごりん`などは通常小数として扱います。
- 変換例：`さんわりにぶごりん → いちとさんわりにぶごりん`
- 変換例：`にわりごぶろくりん → いちとにわりごぶろくりん`
